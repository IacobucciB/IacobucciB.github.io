Veterinaria "Oh My Dog!"

Ruby On Rails, HTML/CSS/JS, Bootstrap, PostgreSQL

Introducción:

Veterinaria "Oh My Dog!" es una página web desarrollada para la veterinaria "Oh My Dog!". Su objetivo es facilitar a los clientes la obtención de turnos para sus mascotas y la adopción de animales abandonados. La página web fue desarrollada por un equipo de estudiantes de la universidad como parte de un proyecto de grado utilizando la metodología ágil Scrum a lo largo de 4 meses.

Ofrece las siguientes funcionalidades:

  • Obtención de turnos

    • Los clientes pueden solicitar turnos para sus mascotas de forma online, seleccionando la fecha, hora y servicio deseado.

  • Adopción de mascotas

    • La página web muestra una lista de animales disponibles para adopción, con fotos, información sobre su raza, edad y características. Los interesados en adoptar una mascota pueden completar un formulario online.

  • Gestión de usuarios

    • Los clientes pueden crear una cuenta para acceder a su historial de turnos y mascotas adoptadas.

  • Administración

    • La página web cuenta con un panel de administración donde el personal de la veterinaria puede gestionar los turnos, las mascotas disponibles para adopción y los usuarios.

Metodología de desarrollo:

El proyecto se desarrolló utilizando la metodología ágil Scrum. Se dividió en sprints de dos semanas, durante los cuales se planificaban, desarrollaban y testaban las funcionalidades de la página web. Al final de cada sprint, se realizaba una demostración del trabajo realizado al cliente.

Tecnologías utilizadas:

  • Ruby on Rails
  • PostgreSQL
  • Bootstrap
  • JavaScript
